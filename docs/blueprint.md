# **App Name**: Gatte Tech Solutions

## Core Features:

- Service Showcase: Showcase the services offered by GATTE Tecnologia in a clear and organized manner using a modern grid layout.
- Promotions Display: Display current promotions and offers with visual emphasis to attract user attention.
- Contact Information: Provide all necessary contact information, including social media links, phone numbers, and email addresses, with a floating WhatsApp button for easy communication.

## Style Guidelines:

- Primary color: Deep dark blue (#0A2239) inspired by the logo, conveying trust and professionalism.
- Background color: Light gray (#F0F0F0), creating a clean and modern look.
- Accent color: Vibrant orange (#FF851B) from the logo's icon point, used for CTAs and highlights.
- Body and headline font: 'Inter' for a modern, objective, and neutral feel.
- Use simple, clean icons for each service to ensure clarity and a professional aesthetic.
- Employ a mobile-first, responsive layout with subtle graphic elements and geometric shapes inspired by the provided institutional image.
- Use subtle transitions and animations to enhance user experience without being distracting.
